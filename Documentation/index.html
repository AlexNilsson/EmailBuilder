How to start

To start project in production mode, you need to run npm start directly from root folder.
Then navigate to http://localhost:8000.
Don't want start this project in NodeJS? No problem, run it on any server!

Need Changes?

If you want to make changes, run npm install && npm run dev and open http://localhost:9000 in your browser for development mode.
All logical functions are in app/app.js and app/builder/builder.html.

Can install on Apache?

Of course, you can install it in any server which you want ... just copy all files from _dist folder in your host root folder!

Dependencies

  • AlertifyJS
  • VueJs
  • TinyMCE
  • Vue Draggable
  • jQuery Powertip

Important: All folders from /app must be accesible from browser!
